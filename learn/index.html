LockPlus A whole new way to make lockscreens

Faster, smarter, and more powerful than any previous LockBuilder. LockBuilder Evo for iOS8 stands in its own class, giving you the ability to create amazing lock screens that you can save, and share. Uses common elements used in a majority of lock screens, and more! Many of the Lockscreens you see in cydia can be created with this one tool.


Getting Started

First, LockPlus must be downloaded from the MacCiti repo. Once it has finished, open the settings app, and find the Cydget section. Tap on the "Lock Cydget Order" section, and tap on LockPlus to enable it. Navigate back to the previous screen, and tap "Respring" in the top left. You will be greeted with a first-time welcome page for LockPlus, giving you further instructions. The information on these pages can be found below.


Menu

After the instructions you can swipe left (from the far right) to display the main menu. Swipe right to close the menu.



Creating

You do not create themes on the device, this allows for much better performance, and we can give more options via the web. You can create themes at LockPlus.us.



Downloading

You can download themes by pressing the download button, and entering a name of a theme. You can download an example theme named LockPlus now to see how this works.



Loading

If you have a theme downloaded you can apply it by tapping load theme in the menu. Swipe left or right to the theme, then swipe up to apply the theme. To delete a previously downloaded theme, tap the trash icon above the theme you wish to delete.
