Theming LockBuilder via WinterBoard

You can change the entire look of LockBuilder's user interface by creating a theme for winterboard. The bundle id is com.JunesiPhone.LockBuilderEvo just add this to any Bundles folder of any theme.

All menu icons can be themed. Included in the bundle you can add a css and js file to theme even more of LockBuilder's interface. An example of what you would use the css and js file for is to change text colors.

Below is a theme template for all icons, the css file and javascript file. It is themed to match Zen8.
