Changing the Background

This brings up the menu to change the background of the theme, to blur the background, to download the background, and to clear the background.

Change the Overlay

An overlay appears above the wallpaper, but behind the other screen elements. It is often used to add special shapes, images, or widgets to a theme, without needing to manually edit the wallpaper image. Through overlays, custom code can be injected at runtime, allowing for widgets. For more info on creating widgets by using svg overlays, see this

Show Elements Panel

Click this button to bring up the selection panel for screen elements. It presents you with the choice between clock elements, weather elements, system elements, and miscellaneous elements.

Save Theme

Saves and uploads the current theme to the server, so it can be downloaded to the cydget. Allows you to choose the name of the theme.

Load Theme

This opens up a page that shows all themes that have been made. You can either click on a theme, or enter the name of a theme, to edit that theme. This loads that theme into the editor, overwriting the previous theme in the editor.

Clear Theme

Clears the current theme in the editor, deleting everything that was previously there.

Load Theme

Load Icon allows you to load any saved, or downloaded themes. Press load and it will load a visual menu of the lockscreens you have. It loads from var/mobile/Libary/LBEvoThemes

Save Theme

Save Icon will allow you to save your theme for later use, or to share with anyone. When you press it, enter a name for your Lockscreen then press ok. Themes are stored in var/mobile/Libary/LBEvoThemes

Save SB

Saves changes to LockBuilder Evo iOS8 SBHTML. The SBHTML must be selected in SBHTML for any changes to show.

Free Theme

Free theme icon opens the listing of free themes you can download and use in lockbuilder. If you find a theme you want click it and you will be taken directly to download it.

Reset LS

Removes current theme loaded to LockBuilder

Reset SB

Removes current theme loaded to SBHTML

Master Reset

Removes everything saved inside LockBuilder. LS and SB.

Select Overlay

Allows you to place an overlay on top of the wallpaper. They are located in var/mobile/Library/LBEvoOverlays

Remove Overlay

Removes current overlay.

Select GroovyLock

Allows you to place a groovylock lockscreen on top of the wallpaper.

Remove GroovyLock

Removes current GroovyLock

Refresh Weather

Pulls new weather information.

Create GL

Takes the current theme and creates a standalone groovylock from it. This addon is available in cydia.

Alert Memory

Alerts current Javascript memory.

Upload Theme

Allows you to send in a lockbuilder theme. First make sure you have saved your theme, then load it. After it is loaded press this button. After it is finished it will ask you to open ifile. There you zip the folder and send to lockbuilder@gmail.com
